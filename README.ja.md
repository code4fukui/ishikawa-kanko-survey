# ishikawa-kanko-survey - 石川県観光データ分析プラットフォーム「Milli」

このリポジトリには、石川県観光調査の自動データ処理パイプラインが含まれています。Googleスプレッドシートから生データを取得し、クレンジング（整形）を行った上で、オープンデータのCSVファイルとして公開します。

## オープンデータ

処理済みの調査データは、3つの地域別データセットおよび統合データセットとして提供されています。

- **全データ:** [all.csv](https://code4fukui.github.io/ishikawa-kanko-survey/all.csv)
- **金沢エリア:** [kanazawa.csv](https://code4fukui.github.io/ishikawa-kanko-survey/kanazawa.csv)
- **加賀エリア:** [kaga.csv](https://code4fukui.github.io/ishikawa-kanko-survey/kaga.csv)
- **能登エリア:** [noto.csv](https://code4fukui.github.io/ishikawa-kanko-survey/noto.csv)

## 仕組み

GitHub Actionsのワークフローが毎日17:25 (UTC) に実行され、データセットを自動的に更新します。

1.  **データ取得:** `download.js` スクリプトが、`list.js` に定義された公開GoogleスプレッドシートのURLから最新のアンケート回答をCSVファイルとして取得します。
2.  **処理・クレンジング:** `make.js` スクリプトが生データを処理します。
    *   3つの地域（金沢、加賀、能登）のデータを統合します。
    *   説明的な長い日本語の列ヘッダーを、短く標準化されたフィールド名にマッピングします。
    *   プライバシーポリシーに同意していない回答を除外します。
    *   タイムスタンプ順にデータを並べ替え、各エントリに一意のIDを割り当てます。
3.  **公開:** 最終的なクレンジング済みCSVファイル（`all.csv`、`kanazawa.csv` など）がリポジトリにコミットされ、GitHub Pagesを通じて公開されます。

## データスキーマ

最終的な `all.csv` ファイルには以下の列が含まれます:

- `ID`: 各回答の一意の識別子
- `回答エリア`: 回答エリア（金沢、加賀、能登）
- `回答日時`: 回答のタイムスタンプ
- `宿泊数（全行程）`: 宿泊数（旅行全体）
- `宿泊数（県内）`: 宿泊数（石川県内）
- `宿泊エリア（県内）`: 宿泊エリア（石川県内）
- `宿泊エリア（県外）`: 宿泊エリア（石川県外）
- `同行者`: 同行者
- `宿泊目的`: 宿泊目的
- `交通手段`: 石川県への交通手段
- `交通手段（施設）`: 施設への交通手段
- `満足度（交通手段）`: 交通手段の満足度
- `満足度理由（交通手段）`: 交通手段の満足度理由
- `一人あたり宿泊費`: 一人あたりの宿泊費
- `一人あたり交通費`: 一人あたりの交通費
- `一人あたり消費額（県内）`: 石川県内での一人あたり消費額（宿泊・交通費を除く）
- `訪問回数`: 施設への訪問回数
- `情報源`: 訪問の情報源
- `訪問施設（前）`: 前に訪問した施設
- `訪問施設（前）自由記入`: 前に訪問した施設（自由記入）
- `訪問施設（後）自由記入`: 次に訪問する施設（自由記入）
- `一人あたり消費額（施設）`: 施設での一人あたり消費額
- `満足度（商品・サービス）`: 商品・サービスの満足度
- `満足度理由（商品・サービス）`: 商品・サービスの満足度理由
- `満足度（施設）`: 施設の満足度
- `満足度理由（施設）`: 施設の満足度理由
- `不便（施設）`: 施設での不便な点
- `不便理由`: 不便な理由
- `NPS`: ネットプロモータースコア (0-10)
- `推薦理由`: 推薦の理由
- `自由意見（施設）`: 施設に関する自由意見
- `リピート意向`: 石川県への再訪意向
- `自由意見（県内）`: 石川県に関する自由意見
- `性別`: 性別
- `生年`: 生年
- `都道府県`: 居住都道府県
- `市区町村`: 居住市区町村
- `世帯年収`: 世帯年収
- `回答動機`: アンケート回答の動機

## 手動での実行

データ処理パイプラインを手動で実行するには、[Deno](https://deno.land/) ランタイムが必要です。

1.  リポジトリをクローンします:
    ```bash
    git clone https://github.com/code4fukui/ishikawa-kanko-survey.git
    cd ishikawa-kanko-survey
    ```
2.  Googleスプレッドシートから最新の生データをダウンロードします:
    ```bash
    deno run -A download.js
    ```
3.  生データを処理し、最終的なCSVファイルを生成します:
    ```bash
    deno run -A make.js
    ```
